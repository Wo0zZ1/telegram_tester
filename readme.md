# Cat test

## How to install?

### 1. Clone the project

```git
git clone https://github.com/Wo0zZ1/telegram_tester
```

### 2. Open the folder

```bash
cd TelegramTester
```

### 3. Install dependencies

```bash
yarn
```

or

```bash
npm install
```

### 4. Start server

```bash
yarn start
```

or

```bash
npm run start
```
